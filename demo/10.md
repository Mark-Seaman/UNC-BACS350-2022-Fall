# Demo 10 - Photo Albums

## Overview

This demo will illustrate how to build an application that works with images.


## User Stories

1 - Users are able to register themselves for a new user account.

2 - Users are able to upload photos and have them displayed.  

3 - Photos are displayed in an image carousel which rotates through all of
the photos.


## Demonstrated Skills

* Photo upload
* Media hosting on server
* Image carousel
