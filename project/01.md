## Project Goals

Setup a git repository stored at Github that will be used for all the code you
will write for BACS 350. Your code repository will be forked from

    https://github.com/Mark-Seaman/BACS350

and named

    https://github.com/<your-name>/BACS350

You will register your Github repo on the Sensei Server at 

    https://shrinking-world.com/bacs350/register

All grading will be done by examining the code in
this course repo.


## Requirements

* Your repo is registered on the Sensei Server
* Your repo is forked from Mark-Seaman/BACS350
* You have created and debugged a Django application with your Profile
